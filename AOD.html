APPLICATIONS OF DERIVATIVES

Learn how to take a derivative and apply that knowledge using these methods!

Critical Points: Critical points are the point(s) at which f ’(x) = 0 or f ‘(x) is undefined. In order to calculate the critical points, one must set the derivative function equal to zero and solve for the variable in the problem.

Increasing/Decreasing Intervals: In order to figure out where a function is increasing and decreasing, one must find where the derivative is positive and where it is negative on a given interval. The best way to do this is to plot out the endpoints and critical points on a number line. Then, plug numbers within each interval into the derivative function to determine where the function is increasing and decreasing.

Relative Extrema: Once you understand the concepts of increasing and decreasing intervals, relative extrema are quite easy. Relative maxima can be found where the derivative function changes signs; when it goes from positive to negative, there is a relative maximum and when it goes from negative to positive, there is a relative minimum.
