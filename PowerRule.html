POWER RULE

Learn how to take a derivative using these methods!

Definition of the Power Rule: A method used to differentiate polynomial functions.

Function Notation: d/dx (xn) = nx(n-1)

Steps to Success:

Step 1: Multiply the current (pre-differentiated) coefficient by the current exponent.
Step 2: Subtract 1 from the current exponent.
Step 3: Simplify, if necessary.
